## Local Development

### Prereqs

- Install [NodeJS](https://nodejs.org/en/)
- Install project dependencies with: `npm i`

### Commands

```bash
# SASS and front/backend JavaScript file watcher
npm run dev

# Create final build files into /dist
npm run build
```
